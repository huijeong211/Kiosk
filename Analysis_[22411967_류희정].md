# 1. Introduction
## 1) Executive Summary
현재 카페 및 음료 매장에서 사용되는 대다수의 키오스크 시스템은 지나치게 복잡한 인터페이스와 불필요한 주문 단계로 인해 오히려 사용자에게 피로감을 주고 있다. 특히 기존 시스템들은 관리자(점주)의 편리함에만 치중하여, 정작 사용자가 직면하는 메뉴 탐색의 어려움이나 복잡한 옵션 선택 등의 문제를 방치하고 있다. 이로 인해 디지털 취약계층은 주문 과정에서 소외되고 있으며, 이는 매장 운영의 효율성 저하로 이어진다.

본 프로젝트에서 개발하는 Beverage Kiosk System은 이러한 기존 시스템의 한계를 극복하고자 한다. 복잡할수록 더 많은 기능을 제공할것이라는 잘못된 설계를 탈피하고, 인지 공학적 접근을 통해 사용자에게 가장 필요한 정보만을 직관적으로 노출함으로써 주문의 진입장벽을 획기적으로 낮춘 시스템을 구축하고자 한다.

## 2) Business Goals
Beverage Kiosk System 시스템의 목적은 사용자가 학습 없이도 즉시 주문을 완료할 수 있는 인터페이스를 제공하는 것이다.
기존 키오스크의 고질적인 문제인 복잡한 뎁스 대신 단 세 번의 터치만으로 결제까지 이어지는 프로세스를 구축하여 사용자 편의성과 매장 회전율을 극대화한다.
또 불필요한 회원가입이나 마케팅 팝업 등 사용자의 집중력을 흐트러뜨리는 요소를 과감히 제거하여 음료 주문이라는 본연의 목적에만 집중하게 함으로써 오주문율을 낮춘다. 이를 통해 매장은 인건비 절감을 넘어, 기술 취약 고객층까지도 아우르는 포용적 서비스를 실현한다.

## 3) Technical Goals
본 시스템은 안정적인 데이터 처리와 직관적인 사용자 경험을 위해 다음과 같은 기술적 목표를 가진다.

Security & Access Control: 재고 수량이나 메뉴 가격 같은 정보는 매우 중요하기 때문에 일반 사용자가 함부로 수정할 수 없도록 설계한다. 시스템 내에서 관리자 권한을 가진 사람만이 해당 데이터에 접근하고 수정할 수 있도록 관리 기능을 제한하여 정보의 안전성을 확보한다.

Algorithm & Performance: 사용자가 카테고리를 선택하거나 메뉴를 장바구니에 담을 때 지연 시간 없이 즉각적으로 실행되도록 설계한다. 각 기능의 특성에 맞는 적합한 알고리즘을 선택하여 데이터 처리 효율을 높이고, 사용자가 기다리는 불편함을 최소화한다.

User Interface (GUI): 기술적인 지식이 없는 사용자라도 화면만 보고 바로 주문을 진행할 수 있도록 누구나 이해하기 쉬운 직관적인 GUI를 제작한다. 불필요한 단계를 줄이고 버튼 위주의 레이아웃을 구성하여 사용자의 편의성을 최우선으로 한다.
